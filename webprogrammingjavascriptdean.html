The button onclick event handler calls the generateEmail function which combines the first name and last name entered. The this keyword in onclick="generateEmail(this.form) refers to the object which contains the JavaScript in which this appears. The button element's object has a form property or .form(?) property which holds the form which surrounds the button. We pass the form object to the generateEmail function by calling generateEmail(this.form).

The outerHTML property returns the text and its tags. The innerHTML property returns the text enclosed in its tags.

Common event handlers attributes: onclick users clicks on an element, onfocus elements gain focus, onchange form control value changes, onmouseover mouse moves over an element, onmouseout mouse moves away from element, onload element finishes loading.

Text control attributes are type, id, placeholder, size, maxlength, value, autofocus, disabled, and readonly. Use type="text" for text control or text box or input box. autofocus specifies the browser engine position the cursor in the text control after the page loads. disabled specifies the text control cannon receive the focus disabled=true. The user can't copy or edit the text control's value. readonly specifies the user highlight the control's value and copy only.
